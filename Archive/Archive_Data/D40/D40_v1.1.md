**스켈레톤 재사용 근거 (SKILL v1.11 "신규 제품 스켈레톤 재사용 규칙")**: 본 파일은 D25_v1.1.md의 Key 목록/섹션 구조를 뼈대로 복제한 뒤 서지컬 교체하여 작성되었다 — D25보다 D40이 더 유사한 기존 제품이 없다고 판단한 근거: D40의 OM/SPS는 문장 수준까지 D25와 거의 동일하다(Milan Endstation, 4.3" 터치스크린, LoadMatch/ArrayProcessing/System check/AmpPresets/EcoMode/Fallback/Override/AutoStandby/AutoWakeup, 심지어 전원 대기상태 소비전력 수치까지 동일). D40은 D25 대비 "High voltage output suitable for driving applicable d&b loudspeakers, including the KSL System"이 추가된 고전압/고출력 변형 모델로 판단되며(SPS "Features and benefits" 절), 실제 값 차이는 채널당 출력(2배 이상), 최대 출력 전압(120V->180V), 출력 노이즈/다이나믹레인지 수치, 네트워크 기본 IP 정도에 국한된다.

이번 소스 집합은 D40 데이터시트(SPS, 4p — 마지막 페이지에 "Architectural specifications" 절 포함, 별도 AE 문서 없이 이 SPS가 AE 역할까지 겸함)와 Reference Manual(OM, 71p) 2개 문서로 구성된다. 웹 3계층 텍스트는 이번 라운드에 제공되지 않아 Raw_Web_Data는 생성하지 않았다. 브랜드 기본 우선순위에 따라 OM을 최우선 출처로 삼고 SPS를 병기한다 — 두 문서의 스펙 표 수치는 실질적으로 100% 일치한다.

본 파일은 D80_v2.0.md/D90_v2.0.md/D25_v1.1.md에서 이미 확정된 최신 구조(d80_specific/d90_specific 섹션 폐지, network 섹션 신설, BTL/PBTL 4개 Key 통합)를 그대로 사용하며, D25 파싱 중 신설된 Max_Output_Current_Peak Key도 이미 포함된 상태로 최초 작성되었다. 이번 라운드에서 발견된 신규 Key는 없다.

### 주석 및 출처 (Notes & Sources)

**출처**: OM p.8-10 "3 Technical specifications" (최우선) / SPS 2-4p

**측정 조건**: 전 채널 구동. Amplification_Gain은 "Linear mode @ 0 dB" 조건(OM/SPS 일치). Output_Delay_Range는 채널별 독립 적용, 최대 10초(OM/SPS 일치).

**[1] Rated_Power_Per_Ch 표 구조 (D25와 완전히 동일한 표기 방식, 값은 D25 대비 확장)**: D40의 OM/SPS는 D25와 동일하게 두 조건의 채널당 출력을 명시한다 — (a) "Output power rating EIA-426B noise CF 12 dB": 8 Ω = 2000 W, 4 Ω = 2400 W(순간/버스트성 정격), (b) "Sine wave 1 kHz, long term, +40°C": 4 Ω = 250 W(연속 정격, 열 제한 조건 — D25와 동일한 250 W, 8 Ω 조건의 별도 long term 수치는 소스에 명시되지 않아 4 Ω 조건에서만 확인된 값임을 각주로 명기). D40의 CF 12 dB 조건 출력은 D25(4 Ω=1800 W, 8 Ω=900 W) 대비 크게 확장되었으나(4 Ω=2400 W, 8 Ω=2000 W), long term 연속 정격(250 W)은 D25와 완전히 동일하다 — 순간 출력 용량만 확장되고 연속 열 제한 정격은 동일한 설계로 판단된다. OM Chapter 3.1의 CF 9 dB/6 dB 상세값(표에는 미채택, 참고용): CF 9 dB(8 Ω = 1150 W, 4 Ω = 1300 W), CF 6 dB(8 Ω = 900 W, 4 Ω = 700 W). 1 kHz burst(20 ms/0 dB, 4 Ω) = 1200 W(110 ms 지속), 1 kHz sine(1채널 구동, 4 Ω) = 2400 W.

**[2] Total_Power_Capability**: D40의 OM/SPS 어디에도 전 채널 동시 구동 시 총 출력 용량을 나타내는 단일 수치가 명시되어 있지 않다. 채널당 정격의 단순 합산은 임의 계산 금지 원칙에 따라 채택하지 않는다. null 처리.

**Max_Output_Voltage_Peak/Max_Output_Current_Peak**: OM/SPS "Maximum output voltage/current 180 Vpeak/35 Apeak" — 전압은 D25(120V) 대비 확장되었으나 전류(35A)는 D25와 동일하다.

**[3] Bridging_Support 및 BTL/PBTL 관련 Key**: D40의 OM("shall be" A&E 절 포함) 및 SPS 전체를 검토했으나 Bridge/BTL/PBTL/bridging 관련 용어나 절차가 전혀 확인되지 않는다 — D25와 동일한 판단(D25_v1.1.md 동일 각주 참조). D40 자신의 원본 문서를 직접 전량 검토한 결과이나, 확정적 부인 문구가 없어 보수적으로 null 유지한다.

### 주석 및 출처 (Notes & Sources)

**출처**: OM p.8 "3 Technical specifications", p.64-66 "10.1 Power supply" (최우선) / SPS 2p

**측정 조건**: Power_Consumption_Idle/Standby는 OM Chapter 3.1 "Current/power draw and thermal dissipation" 표의 230 VAC/50 Hz 대표 조건 기준(다른 전압 조건 - 208V/120V/100V - 은 OM p.11에 별도 표로 제공, 이 스키마는 D80/D90과 동일하게 230V 대표값만 채택). PSU_Type Value는 원문 그대로 채택 — D90과 완전히 동일한 표현.

**[1] Nominal_Current 값 산정**: OM/SPS "Rated mains current (High range) 13 A", "Rated mains current (Low range) 20 A" — D25와 완전히 동일한 값(같은 PSU 플랫폼 공유로 판단). D80(고전압 16A/저전압 30A)·D90(16A/24A)과는 다른 값.

**[2] Power_Consumption_Idle/Standby 값 산정 (OM Chapter 3.1 표, 산술 교차검증)**: D25와 동일하게 OM/SPS 첫 페이지 요약란("Power consumption (typical values) / Standby / Idling / Peak output")은 PDF 레이아웃(다단 표) 추출 시 라벨과 수치가 뒤섞여 신뢰할 수 없었다. 대신 OM Chapter 3.1의 "230 VAC / 50 Hz" 상태별 상세 표를 채택했다 — Off=1.5W, Standby=20.8W, ReadyStandby=52.6W, Eco=103W, Idling=127W(전압×전류×역률 산술로 각 행 자체 검증, D25_v1.1.md 각주 [2]와 동일 방법론). 이 수치는 D25와 완전히 동일하다 — 저전력 대기/제어 회로(Auxiliary Power Supply)가 D25와 같은 설계를 공유하며 출력 파워앰프 스테이지만 확장된 것으로 판단된다. Power_Consumption_Idle=127W, Power_Consumption_Standby=20.8W로 매핑. ReadyStandby(52.6W)/Eco(103W)는 dsp 섹션의 Energy_Saving_Detail 각주에 병기했다.

**Power_Factor**: D40 소스(OM, SPS) 어디에도 D80의 "above 0.9" 같은 명시적 역률 수치가 없어 null 처리했다(D25와 동일 패턴). PFC 기능 자체는 탑재되어 있음(PSU_Type 참조)이 확인되나 구체적 역률 수치는 미확인이다.

**[3] PSU_Mains_Rated_Power**: D40 OM/SPS의 첫 페이지 요약란에도 D25와 동일하게 "Peak output"이라는 항목명이 있으나(위 [2] 참조), PDF 레이아웃 추출 뒤섞임으로 인해 이 항목에 정확히 대응하는 수치를 자신 있게 분리해낼 수 없었다 — OM Chapter 3.1의 상세 전류표에도 "Peak output"이라는 명칭의 단일 대표값 행이 별도로 존재하지 않는다. 무결성 원칙에 따라 D25와 동일하게 null 처리.

### 주석 및 출처 (Notes & Sources)

**출처**: OM p.9 "3 Technical specifications" (최우선) / SPS 2p

**측정 조건**: Frequency_Response는 -1 dB 허용오차, Linear mode 기준(OM/SPS 일치). THD_N/Crosstalk은 unweighted, 20-20 kHz 대역, "4 x 250 W into 8/4 Ω" 조건(long term 정격 기준, OM/SPS 일치) — 부하 조건은 D25와 동일.

**[1] Frequency_Response**: D25/D90과 완전히 동일한 값(35-25000 Hz) — 같은 세대 플랫폼으로 판단.

**[2] THD_N 표기 방식**: D40의 OM/SPS는 "THD+N/Crosstalk < -86 dB/0.005%"(8 Ω), "THD+N (unweighted, 20-20 kHz) < -83 dB/0.007%"(4 Ω)로 D25와 완전히 동일한 수치를 보인다 — 출력 스테이지는 확장되었으나 THD 성능 자체는 D25와 동일한 설계.

**[3] Channel_Separation**: OM/SPS "Crosstalk (20-20 kHz) < -70 dBr"로 D25/D80/D90과 완전히 동일한 수치("less than -70 dBr").

**Output_Dynamic_Range, Noise_Level, Damping_Factor**: D40 소스에는 D25/D80/D90과 마찬가지로 이 세 Key에 정확히 대응하는 스펙이 확인되지 않는다. SN_Ratio_Analog_Input/SN_Ratio_Digital_Input도 D25/D90과 동일하게 단순 dBr 상대값 형식의 S/N ratio 항목이 없어 null 유지 — 실제 값은 절대 노이즈 전압(uV RMS) 기준 데이터 구조이며 아래 Output_Noise_Dynamic_Range_Detail Key에 기록했다.

**[4] Output_Noise_Dynamic_Range_Detail**: OM/SPS "Output noise (BW 20 kHz)/dynamic range (BW 20 kHz, reference 180 Vpk)" 조건 — reference 전압(180 Vpk)은 D40 자신의 Max_Output_Voltage_Peak(180V)와 정확히 일치한다(D90의 reference 210 Vpk = Max 210V 패턴과 동일). 이는 D25_v1.1.md 각주 [4]에서 발견된 이상 사례(D25는 reference 180 Vpk가 D25 자신의 Max_Output_Voltage_Peak 120V와 불일치하고 오히려 D40의 180V와 일치 — D25 문서가 D40 템플릿을 복사하며 갱신하지 않은 오기로 추정됨)를 뒷받침하는 정황 근거다: D40 자신은 정합적이므로, D25의 180 Vpk가 D40에서 비롯된 값일 가능성이 높다. 절대 노이즈 수치 자체도 D25(350/280/220/170 uV, 108/110/112/114 dB)보다 D40이 근소하게 낮은 노이즈/높은 dB(350/250/200/150 uV, 111/114/116/119 dB)를 보여 실제 사양 차이로 판단.
