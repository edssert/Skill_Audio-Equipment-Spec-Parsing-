# K1 (L-Acoustics) — Master Schema

**Category**: speakers | **Brand**: L-Acoustics (LA) | **Schema Phase**: Phase 1 (skeleton origin product for `speakers` category)

이 파일은 `speakers` 카테고리의 최초 제품(스켈레톤 기원)이다. 기존에 참고할 유사 제품 파일이 없으므로(카테고리 신설), "신규 제품 스켈레톤 재사용 규칙"의 적용 대상이 아니라 처음부터 섹션 구조를 새로 설계했다 — `amplifiers/` 카테고리의 섹션 단위 출처 표기/표 순수성/Null Report 등 출력 포맷 규칙은 그대로 계승하되, 섹션 구성 자체는 라인어레이 스피커 엘리먼트의 실제 스펙 항목(음향 성능, 트랜스듀서, 리깅 등)에 맞춰 새로 정의했다.

**핵심 아키텍처 판단**: K1은 원문에 "active WST enclosure"로 표기되어 있으나, 이는 L-Acoustics의 크로스오버 방식 지칭(대역별 개별 앰프 채널 구동)이며 인클로저 자체에는 온보드 앰프가 없다 — "quad-amplified by LA12X"(외부 앰프사용) 및 "Connection to LA amplified controllers" 챕터의 호환 앰프 표, PA-COM 커넥터 핀아웃(외부에서 대역별 앰프 채널을 공급받는 구조)이 근거다. 따라서 AC 전원 입력, 온보드 오디오 입력, 온보드 DSP/네트워크 등 자가발전형(self-powered) 스피커에 해당하는 섹션은 K1에는 원천적으로 적용되지 않는다(비적용) — 이후 Meyer Sound PANTHER(자가발전형) 투입 시 해당 섹션들이 신설되며, K1에는 비적용 사유로 null 반영될 예정이다.

### 주석 및 출처 (Notes & Sources)

**출처**: OM p.135 "K1 specifications", p.15 "Directivity", p.34 "Line source" (Loudspeaker configurations).
**측정 조건**: Usable bandwidth는 -10 dB 기준. Max SPL은 1 m 자유음장 조건, 핑크노이즈(크레스트 팩터 4) 측정, 프리셋 [K1] 적용 기준(원문 각주: "Peak level measured at 1 m under free field conditions using pink noise with crest factor 4 (preset specified in brackets)").
**[1] 프리셋 종속성**: Usable_Bandwidth_Hz와 Max_SPL_Peak 값은 모두 [K1] 프리셋(앰프측 DSP에 로드) 적용을 전제로 한다. K1은 자체 DSP가 없으므로 이 값은 LA12X에 로드된 [K1] 프리셋의 필터링 결과이며, 프리셋 미적용 시 값이 달라질 수 있다.
**Nominal_Directivity_Vertical**: 수치가 아닌 서술형 스펙(어레이 구성 개수와 곡률에 종속) — 원문 그대로 보존.

| Key | Value | Unit |
|---|---|---|
| Compatible_Amplified_Controller | LA12X | - |
| Max_Enclosures_Per_Controller_Output | 2 | count |
| Max_Enclosures_Per_Controller_Total | 2 | count |

### 주석 및 출처 (Notes & Sources)

**출처**: OM p.109 "Connection to LA amplified controllers", "Enclosure drive capacity per amplified controller" 표.
**LA7.16(i) 호환성**: 원문 표에서 K1 행의 LA7.16(i) 컬럼은 "-"(호환 불가/미지원)로 명시됨 — K1은 LA12X 전용. LA7.16(i) 관련 값은 확정적 비존재(해당 없음)이므로 별도 Key를 만들지 않고 Compatible_Amplified_Controller에 LA12X만 기재.
**Max_Enclosures_Per_Controller 표기**: 원문 "per output* / total" = "2 / 2". 각주 원문: "For passive loudspeakers, the value corresponds to the number of enclosures in parallel on the output. For active loudspeakers, the value corresponds to the number of sections in parallel on the output."

## Null Report

이 파일에는 현재 null Key가 없다 — Phase 1 최초 제품으로, 아직 타 브랜드/제품 투입에 따른 소급 null Key가 발생하지 않았다. 이후 GSL8/GSL12(d&b), PANTHER(Meyer Sound) 투입 시 각 제품 고유 섹션(예: PANTHER의 power_supply, audio_input, network 등 자가발전형 전용 섹션)이 K1에 비적용(null) 사유로 소급 반영될 예정이다.

**총계**: null 0건 (Phase 1 시점 기준).
