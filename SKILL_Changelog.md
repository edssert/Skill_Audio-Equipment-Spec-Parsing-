# SKILL 버전 이력 (Changelog)

이 파일은 `SKILL_v*.md`의 전체 버전 이력을 보관하는 append-only 전역 문서다. SKILL.md 본문에는 이 이력을 싣지 않는다 — 스킬 로드 시 컨텍스트를 소모하지 않도록 분리했으며, 과거 버전의 상세 배경이 필요할 때만 이 파일을 읽는다. 각 버전의 원문 전체는 `Archive/Archive_Skill/`의 구버전 파일에 그대로 남아 있다.

운영 규칙: SKILL.md가 새 버전으로 갱신될 때마다 이 파일 하단 표에 해당 버전의 항목을 추가한다(기존 항목은 수정하지 않는다). 이 파일 자체는 버전 접미사 없이 제자리에서 추가만 하는 예외 문서다(CLAUDE.md와 동일한 취급).
